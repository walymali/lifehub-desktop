Viral Twitter/X Thread High Impact

Write a 10-tweet thread about [TOPIC] for [TARGET AUDIENCE]. Start with a bold hook that creates curiosity. Each tweet should be under 280 characters. LinkedIn Thought Leadership Post

TikTok/Reels Script (60 seconds)

Engagement Bait Post

SEO Blog Post (2000 words) Best Seller

Write a comprehensive 2000-word blog post targeting the keyword "[KEYWORD]". Include: an engaging H1 title with the keyword, meta description (155 chars), introduction with a hook, 5-7 H2 subheadings, H3s where needed, bullet points, a FAQ section (4 questions), and a conclusion with CTA. Use the keyword naturally 8-12 times. Write for [TARGET AUDIENCE] at a [beginner/intermediate/advanced] level. Tone: [professional/casual/authoritative].

Listicle Post

Write a listicle blog post: "[NUMBER] Best [ITEMS] for [AUDIENCE] in 2026". For each item include: name, one-line description, 3 pros, 1 con, price (if applicable), and who it's best for. Add an intro explaining your selection criteria and a comparison table at the end. Target keyword: [KEYWORD].

How-To Tutorial

Write a step-by-step tutorial on "[HOW TO DO X]". Include: numbered steps with screenshots suggestions, pro tips in callout boxes, common mistakes to avoid, estimated time for each step, tools/resources needed, and a troubleshooting FAQ at the end. Target keyword: [KEYWORD]. Make it so clear a beginner can follow.

Blog Post Outline Generator

Create a detailed outline for a blog post about "[TOPIC]" targeting "[KEYWORD]". Include: 3 title options (with keyword), meta description, intro hook, 6-8 H2 sections with 2-3 bullet points each describing what to cover, internal linking opportunities, CTA placement, and word count estimate per section.

Comparison Post (X vs Y)

Write a detailed comparison post: "[PRODUCT A] vs [PRODUCT B]: Which is Better in 2026?" Cover: overview of each, feature-by-feature comparison (make a table), pricing, pros/cons, use cases (who should choose what), and your final verdict. Be balanced but give a clear recommendation. Target keyword: [KEYWORD].

Case Study Blog Post

Write a case study blog post about how [CLIENT/COMPANY] achieved [RESULT] using [PRODUCT/METHOD]. Structure: Challenge (what problem they faced), Solution (what was implemented), Results (specific numbers), Key Takeaways (3-5 lessons). Include quotes, data visualizations suggestions, and a CTA.

Pillar Page / Ultimate Guide

Create an outline for an ultimate guide on "[BROAD TOPIC]" (5000+ words). Include: table of contents, 10-12 main sections, sub-sections for each, where to link to supporting blog posts, downloadable resource ideas, and FAQ section. This should be the most comprehensive resource on this topic online.

Blog Intro Variations (5 styles)

Write 5 different blog post introductions for an article about "[TOPIC]". Style 1: Start with a shocking statistic. Style 2: Open with a personal story. Style 3: Ask a provocative question. Style 4: Paint a future scenario. Style 5: Start with a bold, contrarian statement. Each should be 80-120 words and naturally include "[KEYWORD]".

FAQ Schema Content

Write 10 FAQ questions and answers about [TOPIC] optimized for Google's FAQ rich snippets. Each answer should be 50-100 words, include the target keyword naturally, provide genuine value, and be structured for featured snippet potential. Main keyword: [KEYWORD]. Related keywords: [LIST].

Content Update / Refresh

I have an existing blog post about [TOPIC] that's losing rankings. Here's the current content: [PASTE CONTENT OR URL]. Analyze it and suggest: 1) Updated statistics/data, 2) New sections to add, 3) Outdated info to remove, 4) Better title options, 5) Internal linking opportunities, 6) Featured snippet optimization.

Roundup / Expert Quotes Post

Create a framework for an expert roundup post: "[NUMBER] [EXPERTS] Share Their Top [TOPIC] Tips for 2026". Write: intro, the main question to ask experts, 10 fictional but realistic expert responses with names/titles/companies, key themes summary, and conclusion. Include how to actually reach out to real experts.

News Hijacking / Trend Post

Write a blog post connecting [TRENDING NEWS/EVENT] to [YOUR NICHE]. Title should include both the trend and your niche. Explain the news briefly, analyze its impact on [INDUSTRY], provide 3-5 actionable insights for your audience, and include your expert perspective. Speed and relevance are key.

Beginner's Guide

Write "The Complete Beginner's Guide to [TOPIC]" for someone who knows nothing about it. Define all jargon, use analogies to explain complex concepts, include a "start here" action plan, recommend beginner resources/tools, and address common fears/misconceptions. 1500-2000 words. Tone: encouraging and patient.

Meta Description Generator

Write 5 meta description variations for a blog post titled "[TITLE]" targeting "[KEYWORD]". Each must be: under 155 characters, include the primary keyword, have a clear value proposition, create curiosity or urgency, and include a soft CTA. Rank them by likely click-through rate.

Blog Post Conclusion + CTA

Write 3 conclusion variations for a blog post about [TOPIC]. Each should: summarize key takeaways (3 bullet points), reinforce the main message, and include a different CTA: 1) Newsletter signup, 2) Product/service promotion, 3) Related content suggestion. Each conclusion: 100-150 words.
